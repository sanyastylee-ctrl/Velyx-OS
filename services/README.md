# Сервисы Velyx OS

Здесь размещаются системные и session-сервисы, которые не должны быть спрятаны внутри UI-приложений.

## Security-critical модули

- `update-engine` — проверка, применение и откат обновлений
- `recovery-service` — сценарии восстановления, snapshots, rollback coordination
- `permissions-service` — выдача и проверка разрешений, privacy ledger
- `compatibility-manager` — безопасный запуск Windows-приложений в управляемой среде

## Базовое правило

UI не выполняет привилегированные операции напрямую. UI только обращается к сервисам через четкие контракты.
